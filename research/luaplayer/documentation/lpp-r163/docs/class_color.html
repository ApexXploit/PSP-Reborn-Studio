Return a table with the following members :
-"r" for the Red value of the Color.
-"g" for the Green value of the Color.
-"b" for the Blue value of the Color.
-"a" for the Alpha value of the Color.
The function must be called from an instance of Color with a colon.
Example :

 c = Color.new(255,0,0)
 t = c:colors()
 screen:print(0, 0, "Red value : "..t.r)
