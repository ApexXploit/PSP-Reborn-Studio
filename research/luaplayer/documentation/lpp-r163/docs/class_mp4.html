table Mp4:getInfo ( )

Return a table that contais the video info.
The table elements are the following :

  • frame_count : the total video frames.
  • width : the video frames with.
  • height : the video frames height.
  • type : video type.
  • video_rate : video rate.
  • video_scale : video scale.
  • audio_type : video audio_type.
